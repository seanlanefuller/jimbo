# Jimbo

A simple SQLite data management project.

## File Structure

- `jimbo.sqlite`: The main database file.
- `configure.txt`: Configuration file pointing to the database.
- `logFile.log`: Application runtime logs.

## Setup

Ensure that `configure.txt` points to the correct location of `jimbo.sqlite`.
